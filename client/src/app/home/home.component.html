Welcome to Dev-Duel!

Dev Duel is a fun and interactive web app where two GitHub users go head-to-head.

Enter two usernames, hit “Duel,” and watch as animated characters battle it out, informed by each user's GitHub stats.

This app is built with an Angular frontend and a Node.js/Express backend, and is styled using TailwindCSS. The app is deployed using Render and the code is tracked on Github.
