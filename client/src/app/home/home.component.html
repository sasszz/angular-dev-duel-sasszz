Welcome to Dev-Duel

Dev Duel is a fun and interactive web app where two GitHub users go head-to-head in a stat-based showdown.

Enter two usernames, hit “Duel,” and watch as animated characters battle it out, all powered by real GitHub data like stars, followers, and top repos.

Built with an Angular frontend and a Node.js/Express backend, the app fetches data from the GitHub API and visualizes it with custom graphics and animations.
